NRC Roleplay.

NRC is een server op FiveM. De naam NRC staat voor Nederlandse Roleplay Community. Deze server staat helemaal in het teken van roleplay, ook word hier heel streng naar gekeken... Mensen die dus nog nooit hebben gespeeld op een RP server, dan raad ik je aan om even door te lezen.

Voor het spelen van FiveM heb je GTA V nodig. Als je onze server wil spelen is er een link naar de discord bovenaan.

Het Politie Korps

In de stad hebben wij een groot politie korps. Als politie patrouilleer je in de stad en bescherm je burgers tegen criminaliteit. Je gaat naar winkel en bank overvallen en achtervolgt criminlelen met hoge snelheden over de snelweg. Het politie korps heeft een eigen Politie Panel. Wil jij ook politie worden in onze stad? Wat is NRC

NRC is een server op Fivem. Deze server staat helemaal in het teken van roleplay, ook word hier heel streng naar gekeken... Mensen die dus nog nooit hebben gespeeld op een RP server, dan raad ik je aan om even door te lezen.

Wat is RP?

Als je NRC joined is het de bedoeling dat je een karakter aanmaakt waarmee jij gaat roleplayen. Dit betekent dat je in je rol blijft als je in de server zit. Denk aan anderen en zorg voor leuke scenario's i.p.v te focusen op geld.

Solliciteren

Heb jij interesse om te solliciteren voor: Politie, Ambulance, ANWB, Advocaat of Makelaar? Voor bepaalde beroepen gelden kleine regels. Je moet voor het worden van politie agent minimaal 16 jaar zijn. Voor elke beroep zul je een vragenlijst op moeten sturen. Dat kan via de knop hieronder. Je kunt reactie verwachten via Discord/Steam/E-mail. Veel succes iedereen!

Als je een reactie ontvangt op je sollicitatie ben je door naar de volgende stap, dat is het sollicitatie gesprek. Dit gesprek ga je voeren met de desbetreffende persoon van het beroep waarop je solliciteert. Als dit gesprek goed verloopt ben je aangenomen! Vanaf dat moment kun je in de stad met je baan beginnen!
